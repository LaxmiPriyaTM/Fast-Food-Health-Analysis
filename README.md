# Analysis of Fast Food Consumption and Its Impact on Health

## 📌 Project Overview
This project analyzes how fast food consumption and lifestyle factors impact health indicators such as BMI, energy level, and overall health score.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Fast Food vs BMI
- Calories vs BMI
- Physical Activity vs Health
- Sleep vs Energy
- Correlation Heatmap

## 💡 Key Insights
- Higher fast food consumption increases BMI
- Physical activity improves health score
- Proper sleep increases energy levels

## 📂 Dataset
The dataset contains lifestyle and health-related attributes including food habits, BMI, sleep, and activity.

## 🚀 Conclusion
Lifestyle choices significantly influence health outcomes. Data analysis helps identify these patterns.
